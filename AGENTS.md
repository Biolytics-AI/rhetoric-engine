# Biolytics Rhetoric Engine

Use this repository as an installable agent skill package for presentation cognition. For substantial presentation work, start with:

- `skills/rhetorical-orchestrator/SKILL.md`

Then route to the named stage skills under `skills/`. The core rule is simple: do not produce a deck until the user has approved the intent, insight, argument spine, and slide thesis map.

For install instructions across Codex, Claude Code, Gemini CLI, and generic agents, see `INSTALL.md`.
