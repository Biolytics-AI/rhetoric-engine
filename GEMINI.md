# Biolytics Rhetoric Engine

Use this extension when the user wants to create, improve, critique, rehearse, or compile a presentation, talk, slide deck, or slide-based argument.

Start with `skills/rhetorical-orchestrator/SKILL.md`. Treat that file as the routing gate. Do not draft or compile slides until the upstream artifacts required by the orchestrator are explicit and approved:

- approved intent
- approved insight
- approved argument spine
- approved slide thesis map

When a routed stage applies, read the matching skill file under `skills/<skill-name>/SKILL.md` and follow it directly. Preserve the user's point of view; the model should sharpen and pressure-test the user's thinking, not replace it with a generic deck.
